# Archive

The Archive is where inactive things live.

It contains things that are finished, outdated or currently not relevant anymore — but might still be useful in the future.

## Purpose

The Archive exists to keep your active workspace clean.

Moving something here does not mean deleting it.

It simply means:

"This is not something I need to think about right now."

## What belongs here?

Examples:

- Finished projects
- Old routines
- Previous versions of documents
- Ideas that are no longer active
- Knowledge you want to keep but rarely use

## What does not belong here?

Do not use the Archive as a second Inbox.

If something is still active, keep it in Active.

If something has no value anymore, delete it.

## Remember

The Archive is not a graveyard.

It is a place where finished things can rest.

Keep your Active folder focused on what matters now.
