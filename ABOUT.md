# About

## Issues and Pull Requests

I don't really expect this repository to receive many issues. It's a highly opinionated project that exists primarily to match my own workflow, thoughts, and preferences.

That said, I intentionally keep GitHub Issues enabled. If you spot a typo, have a small suggestion, or notice something that could be improved, feel free to open an issue. English isn't my native language, so corrections are always appreciated.

If you'd rather fix it yourself, feel free to submit a pull request.

Pull requests that aim to change the core design principles, philosophy, or overall structure of this project will most likely be closed without discussion. This repository is intentionally opinionated and provided **as is**. If you prefer a different approach, I encourage you to fork it and adapt it to your own needs.

### AI Slop

I'm not interested in fully AI-generated issues or pull requests.

I use AI myself from time to time—mostly to improve wording, grammar, or spelling. That's perfectly fine. What I'm trying to avoid are low-effort, fully automated contributions generated by an agent without meaningful human review or thought.

If you're using AI as a writing assistant, that's completely fine. If you're using it to replace your own thinking, please don't contribute it here.

## Commercial Use

If you use this reference model in a commercial environment and find it valuable, I'd appreciate it if you supported my work in some way, whether that's through Ko-fi, GitHub Sponsors, or by reaching out if you have another idea.

That said, the project is licensed under the MIT License. There are no additional restrictions on commercial use.

## Non-Commercial Use

If you find this project useful or it helped you build your own "second brain," consider leaving a ⭐ on GitHub. It helps others discover the project and lets me know that people are finding it useful.

As mentioned above, financial support is always appreciated but never expected.
