### Active

This is the place for your tasks and projects.

Time to take action — but don't overwhelm yourself.
Limit the amount of active things you keep here.

**Important:** Keep your routines healthy before adding more projects.

#### Projects

Projects are "one-time" things with a defined outcome.

Put active projects here.
As soon as you finish one, move it to the Archive.

Use a subfolder for each project and give it a strong, memorable name.

#### Routines

An important perspective on routines:

Think of routines as never-ending projects with periodic renewal.

This is where recurring tasks belong.

For example: `carservice.md` in the yearly folder.

Use this file to remember everything needed:
- When it needs to happen
- What needs to be done
- Important details like telephone numbers, addresses or instructions

Use one file per routine.
Describe it as if you had forgotten everything about it.

Once a recurring task becomes obsolete, move it to the Archive.
